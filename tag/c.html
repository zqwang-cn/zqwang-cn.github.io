c语言++操作符与函数调用的问题

假设有如下c函数：

void func(int a, int b, int c)
{
    printf("%d,%d,%d\n",a,b,c);
}

使用如下方式调用：

int i=0;
func(i++, i++, i++);

希望的输出是：

0,1,2

但是实际输出是：

2,1,0

分析原因，应该是由于c语言的参数压栈顺序为从右向左，因此实际执行的过程是：

int i = 0;
int c = i;
i = i + 1;
int b = i;
i = i …
